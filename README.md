#  Amazon Webpage Clone

This project is a static frontend clone of the Amazon website created using only HTML and CSS.  
It focuses on recreating the visual layout and structure of Amazon’s homepage for learning and practice purposes.

---

##  Project Overview

The Amazon Webpage Clone is a simple frontend project that demonstrates how real-world websites can be built using basic web technologies. This project does not include backend functionality or dynamic features.

---

##  Features

- Amazon-style homepage layout  
- Product sections and banners  
- Clean UI design  
- Static frontend webpage  

---

##  Technologies Used

- HTML  
- CSS  

---

##  How to Run

1. Download or clone the repository.
2. Open the `index.html` file in any web browser.

That’s it — no installation required.
